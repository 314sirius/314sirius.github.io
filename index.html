Methodological Principles

Multicollinearity Control

Redundant features lead to unstable models and misleading feature importance. I enforce strict collinearity checks and prioritize continuous physical variables over repetitive categorical classifications.

Strict Regularization

A high concordance index or near-perfect accuracy on a training set is often an indicator of overfitting. I limit model capacity and tree depth to capture stable, underlying physical trends rather than data noise.

Leakage Prevention

Standard random split validations fail when data contains grouped occurrences (e.g., simultaneous satellite launches). I implement group-aware splitting to evaluate how models generalize to entirely unseen epochs.
